# Ray Engine

A simple raycasting-based game engine written in C using the Raylib library.
This project is created as a learning exercise to understand how classic
raycasting engines work at a low level.

---

## Installation and Run (Linux)

### Requirements
- GCC
- Make
- Raylib

### Build
First, clone the repository and enter the project directory:
```sh
git clone https://github.com/etzjinx/ray-engine.git
cd ray-engine

#Then build the project using:
make


#After building, run the game with:
./game
